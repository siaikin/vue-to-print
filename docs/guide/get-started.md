# Getting Started

## Installation

### Prerequisites

- [Vue.js](https://vuejs.org/) 3.x

::: code-group

```shell [npm]
$ npm install vue-to-print --save
```

```shell [pnpm]
$ pnpm add vue-to-print
```

```shell [yarn]
$ yarn add vue-to-print
```

:::
